# 🏥 Hospital Appointment Booking System

## 📌 Project Overview

The Hospital Appointment Booking System is a web-based application that enables patients to book appointments with doctors online. It provides an easy and secure platform for patients to register, log in, view doctors, book appointments, and manage their profiles. The system also allows administrators to manage doctors, patients, and appointments efficiently.

---

## 🚀 Features

### Patient Module
- User Registration and Login
- View Available Doctors
- Book Appointments
- View Appointment History
- Update Profile

### Admin Module
- Secure Admin Login
- Manage Doctors
- Manage Patients
- View and Manage Appointments
- Dashboard

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Indirapriyadarshinik/hospital-appointment-booking.git
```

### Install Dependencies

```bash
npm install
```

### Start the Backend Server

```bash
npm start
```
